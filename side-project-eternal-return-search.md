# 이터널리턴 전적검색 사이트

이터널리턴 플레이어를 위한 **간단한 전적 조회 웹사이트**입니다.

## 요약

- 닉네임 기반으로 전적을 빠르게 확인할 수 있습니다.
- 최근 플레이 기록과 기본 통계를 한눈에 보기 쉽게 구성했습니다.
- 가볍게 접속해서 필요한 정보만 확인하는 것을 목표로 만들었습니다.

## 프로젝트 상태

- 현재 이 프로젝트는 **개발 중단(아카이브 상태)** 입니다.
- 링크는 참고/기록용으로 유지하고 있습니다.

## 사용 링크

- 서비스 바로가기: [이터널리턴 전적검색 사이트](https://eternal-return-search.vercel.app/)

## 한 줄 소개

게임 들어가기 전에 전적을 빠르게 확인하고 싶을 때 쓰기 좋은 프로젝트입니다.
